## Quick start

**Run Environment: Java11, Node 12, Maven3**


**Run back end server**

```bash
mvn clean package
```

**Then**

```bash
java -jar target/security 0.0.1-SNAPSHOT.jar
```

**Run front end server**

```bash
npm install
```

**Then**

```bash
npm run serve
```
